# Poker Bot

A comprehensive poker bot project featuring a custom game engine, GTO solver integration, strategy database, and real-time AI capabilities.

## Project Structure

- **game-engine**: Core poker game logic and engine implementation (C++).
- **gto-solver**: Tools for Game Theory Optimal (GTO) strategy calculation and analysis.
- **real-time AI**: Real-time decision making and AI components.
- **strategy-database**: Database for storing and retrieving poker strategies and hand histories.
